# IPC2026 — 12th International Conference on Innovative Production and Construction

**November 19–20, 2026 | Melbourne Convention and Exhibition Centre, Melbourne, Australia**

Conference website for IPC2026, co-hosted with DBI2026 — Decarbonising Building Industry International Conference 2026.

## Structure

- `index.html` — Home / Welcome
- `about.html` — About & Conference Leadership
- `themes.html` — Conference Themes
- `submissions.html` — Submission Guidelines & Important Dates
- `committee.html` — Scientific Committee
- `venue.html` — Conference City & Venue
- `contact.html` — Contact
- `css/style.css` — Shared stylesheet
- `js/main.js` — Shared JavaScript (mobile nav, countdown, search)

## Deployment

This site is deployed via GitHub Pages from the `main` branch root.
